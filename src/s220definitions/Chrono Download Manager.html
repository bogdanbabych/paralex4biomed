Welcome!
Before you start, please take these two extra steps to ensure better experience with Chrono.

1. Turn off Chrome's Save As dialog so that you won't be asked where to save file for every download. This can be done by clearing the checkbox before chrome://settings > Downloads > Ask where to save each file before downloading.

2. Allow Chrono to access file URLs for fast file preview, by ticking the checkbox Allow access to file URLs on chrome://extensions.

If you have any question, please visit our Help page and FAQ page.
